Aprendizados

Estudamos o sistema operacional que passaríamos a trabalhar e sobre diversos comandos utilizados no Bash. Começamos a entender como funcionava o Git e o Github para poder trabalhar com versionamento de códigos e armazená-los em reposítórios online. A prendemos em seguida sobre todo o funcionamento da Internet. Por fim, mais no final do bloco começamos a aprender sobre HTML e CSS, para começar a fazer nossas primeiras páginas web como essa e estilizá-las lindamente com o CSS.
